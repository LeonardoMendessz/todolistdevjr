# 📋 To-Do List API

Uma API REST simples para gerenciamento de tarefas (To-Dos), desenvolvida com Spring Boot e WebFlux.

## 🚀 Funcionalidades

- ✅ Criar uma nova tarefa  
- 📄 Listar todas as tarefas  
- ✏️ Atualizar tarefa existente  
- ❌ Deletar tarefa  

## 🛠️ Tecnologias Utilizadas

- Java 17  
- Spring Boot  
- Spring WebFlux  
- Maven  
- MySQL
- Swagger
- Bean Validation
